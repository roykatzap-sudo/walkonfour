## 🎯 הממצא המכריע

**אין דומיננט בגוגל ישראל בנושא כלבים.** SERP מפוזר בין 10-15 אתרים בינוניים (chayuta, barking, wisedog, biopet, petsi, purina, dog-center). זה אומר: **יש פתח אמיתי**.

**הזהב שלנו = תוכן מקומי-גיאוגרפי**, לא תוכן גזעים גנרי:
- 621 גינות עם קואורדינטות = אף אחד לא יודע לעשות
- 38 מסלולי טיול עם מטא-דאטה (אורך/קושי/מים/צל) = ייחודי
- קהילה אמיתית עם דירוגים = יתרון תחרותי

**5 פערים אצל המתחרים שניקח:**
1. אף אחד לא עושה Schema.org ברצינות
2. אף אחד לא משלב FAQ אמיתי בעמודי גזע
3. אף אחד לא מתאם את התוכן לישראל (מחירים ₪, חוקים, אקלים)
4. אפס מחשבונים אינטראקטיביים
5. אפס עמודי השוואת גזעים

### גוף הדף - 10 סקשנים בסדר הצפוי

המתחרים כולם עושים את אותו הדבר - וזה לא מקרה. גוגל אוהב כשהמבנה צפוי.

1. **תיאור כללי** (200 מילים)
2. **★ FAQ ראשי - 5 שאלות מ-PAA** (Schema FAQPage) - **כאן ולא בסוף**
3. **תכונות אופי + רמות אנרגיה/אגרסיביות** (כבר יש לנו!)
4. **"האם מתאים לי?"** - Money section
5. **בריאות ומחלות נפוצות**
6. **אימון ואילוף**
7. **טיפוח (פרווה, ציפורניים, רחצה)**
8. **תזונה והאכלה**
9. **★ "X בישראל"** - בלוק ייחודי: אקלים, חוקים, מחירים ₪ - **אף אחד לא עושה!**
10. **היסטוריה ומוצא** (קצר, בסוף - low intent)

## 🔧 Schema.org - הגרף החדש

זה ה-JSON-LD שיוזרק לכל דף גזע. מבוסס על המחקר העדכני (FAQPage שווה לעשות אבל פחות חזק, INP נכנס כ-ranking factor):

```json
{
  "@context": "https://schema.org",
  "@graph": [
    {
      "@type": "Article",
      "@id": "https://walkonfour.org/breeds/golden#article",
      "headline": "גולדן רטריבר - מדריך גזע מלא",
      "image": [hero_1x1, hero_4x3, hero_16x9],
      "datePublished": "...",
      "dateModified": "...",
      "author": {"@type": "Person", "name": "..."},
      "publisher": {"@id": "...#organization"},
      "inLanguage": "he-IL",
      "about": {"@id": "...#breed"}
    },
    {
      "@type": "Thing",  // אין schema.org/Breed - משתמשים ב-Thing
      "@id": "...#breed",
      "name": "גולדן רטריבר",
      "alternateName": ["Golden Retriever", "גולדן"],
      "sameAs": [
        "https://he.wikipedia.org/wiki/גולדן_רטריבר",
        "https://en.wikipedia.org/wiki/Golden_Retriever",
        "https://www.wikidata.org/wiki/Q38904"
      ]  // ★ זה ה-trick! מקשר לידע של גוגל
    },
    {
      "@type": "FAQPage",
      "mainEntity": [
        {"@type": "Question", "name": "...", "acceptedAnswer": {...}}
      ]
    },
    {
      "@type": "BreadcrumbList",
      "itemListElement": [...]
    },
    {
      "@type": "ImageObject",
      "contentUrl": "...",
      "creator": {...}
    }
  ]
}
```

**הטריק הנדיר:** `sameAs` ל-Wikipedia/Wikidata. מעט אתרי גזעים עושים את זה - מקשר אותנו ל-Knowledge Graph של Google.

## ⚠️ אזהרות Cannibalization (שעלולים לחנוק עצמנו)

1. **`/breeds/labrador` ↔ מאמר "כמה עולה לברדור"** - חייב הפרדה ברורה
2. **דף הגזע ↔ מדריך הגזע ב-`/articles`** - canonical או איחוד
3. **השוואה "לברדור או גולדן" ↔ 2 דפי הגזעים** - השוואה לא חוזרת על המאפיינים
4. **"כלב בקיץ" ↔ "חופי כלבים" ↔ "טיולים בגולן בקיץ"** - pillar + sub-pages
5. **מאמרים על קיא/חוסר תיאבון/אכילת דשא** - אחד pillar, שניים sub

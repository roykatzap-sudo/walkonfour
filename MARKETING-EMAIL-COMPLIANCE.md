# צ'קליסט שליחת דיוור - חוק הספאם (סעיף 30א לחוק התקשורת)

לפני שליחת כל מייל שיווקי לרשימת ההמתנה:

## למי שולחים
- **דיוור שיווקי** (הצעות/מבצעים): רק לנמענים עם `marketing_consent = true`.
  ייצוא: Supabase → Table Editor → טבלת `waitlist` → סנן `marketing_consent = true`.
- **הודעת שירות** ("נפתחנו, בואו"): מותר לכל הרשימה - הם נרשמו כדי שנעדכן (לא "פרסומת").

## תוכן המייל (חובה כשיש תוכן שיווקי)
1. המילה **"פרסומת"** בנושא/כותרת המייל.
2. זיהוי שולח: **קהילה על ארבע** + דרך קשר (כתובת מייל + טלפון או כתובת פיזית).
3. קישור הסרה בתחתית כל מייל:
   `https://walkonfour.org/unsubscribe?email={{EMAIL}}`  (החלף {{EMAIL}} במייל הנמען)
4. ההסרה חינמית, בקליק, ומתבצעת מיד (החוק: עד 3 ימי עסקים).

## אחרי הסרה
- מי שלחץ הסרה → `marketing_consent` מתעדכן אוטומטית ל-false. אל תשלח אליו שוב.
- לפני כל קמפיין - ייצא מחדש את הרשימה המסוננת.

## לפני הקמפיין הראשון
- הרץ את תבנית המייל מול עו"ד לאישור סופי.
